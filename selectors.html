CSS Selectors

X + Y

only the element after

hr + p

This p should be green

This p should not

X > Y

only the child of

div > ul
  • ul child
  • not child ul

X ~ Y

the element after (less strict)

p ~ div

The next items will change

This div is green
This div is also green

Not this p

X[title]

Applies to [item] only

div[title]
Div with title is green
Div w/o title is not
